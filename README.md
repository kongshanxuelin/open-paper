# Description


> This is a papaer system ,written by [Vue2](https://github.com/vuejs/vue) and  [JSPTagEx](https://github.com/kongshanxuelin/jsptagex)，[Live Demo](https://wx.sumslack.com/)


# Requirements
------------

* Bash(git、ssh)
* npm
* jdk8+

## Installation

``` bash

# 安装依赖
npm i

# 运行
npm run dev

# build for production with minification
npm run build

```
